# ZestSSO V1.1 推进路线图

> V1.0.0 已达成实验室验收全 PASS。V1.1 聚焦 **真实业务联调、生产硬化、功能补全**。

## 目标

| 维度 | V1.0 | V1.1 目标 |
|------|------|-----------|
| RP 单点登出 | 代码 + 配置就绪 | **ZestFlow/ZestLLM 实机 Back-Channel 验证** |
| 联调可重复性 | 手工 curl | **内置测试 RP + `sso-backchannel-e2e.ps1`** ✅ |
| 生产部署 | 文档 + Helm | **application-prod 清单 + 上线 SOP 实跑** |
| 告警/短信 | Admin API + 日志模拟 | **真实通道发送（凭据就绪后）** |
| 合规 | 材料包 | **外部渗透/等保（需机构）** |
