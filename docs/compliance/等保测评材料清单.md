# 等保测评材料清单（ZestSSO）

> 适用于网络安全等级保护 **二级 / 三级** 测评准备。具体控制项以测评机构最新要求为准。

## 一、单位基础材料

- [ ] 营业执照复印件
- [ ] 定级报告 / 备案证明
- [ ] 网络拓扑图（含 ZestSSO、MySQL、Redis、Nginx、业务 RP）
- [ ] 机房 / 云资源说明（区域、可用区、安全组）

## 二、ZestSSO 技术材料

- [ ] 系统架构说明（`docs/architecture.md`）
- [ ] 部署手册（`docs/deployment.md`）
- [ ] 数据流图：用户登录 → Token 签发 → RP 校验 → 登出吊销
- [ ] 账号清单：Admin、数据库、Redis、OAuth 客户端（脱敏）
- [ ] JWT 密钥管理说明（文件权限 / K8s Secret / DB 轮换）
- [ ] 备份与恢复演练记录（`scripts/backup-prod.*` 执行截图）
- [ ] 审计日志样例（登录、改密、客户端变更、Token 吊销）
- [ ] 漏洞扫描 / 渗透测试报告（见本目录渗透文档）

## 三、制度与记录（模板见安全管理制度模板.md）

- [ ] 信息安全管理制度
- [ ] 账号及权限管理规定
- [ ] 审计日志管理规定（保留 ≥180 天）
- [ ] 变更与发布管理制度
- [ ] 应急响应预案（含 SSO 不可用时的业务降级）
- [ ] 培训记录（管理员 MFA、Passkey 使用）

## 四、对照技术要点（ZestSSO 能力映射）

| 等保关注点 | 证明材料 |
|------------|----------|
| 身份鉴别 | MFA、Passkey、密码策略、登录锁定 |
| 访问控制 | RBAC、租户隔离、OAuth Scope |
| 安全审计 | `sso_audit_log` 导出样例 |
| 通信保密 | HTTPS 配置、TLS 证书 |
| 数据完整性 | BCrypt 密码、RS256 JWT |
| 备份恢复 | 备份脚本 + 恢复演练记录 |

## 五、测评当天配合事项

1. 提供测评专用只读 Admin 账号（测评后禁用）
2. 演示：正常登录、失败锁定、MFA、Admin 审计查询、强制登出
3. 演示：Back-Channel Logout 联调记录（ZestFlow/ZestLLM E2E 报告）
4. 提供 `scripts/acceptance.ps1` 最近一次全绿报告

## 六、常见整改项预防

| 风险 | 预防 |
|------|------|
| 默认口令 | 上线前更换 admin、MySQL、OAuth client secret |
| 明文传输 | 全站 HTTPS，HSTS |
| 日志不足 | 开启审计导出定时任务 |
| 弱密码策略 | Admin 启用密码策略配置 |

正式测评需委托具备资质的测评机构，本清单仅作准备索引。
